# Triage Labels

スキル群は 5 つの正典 triage 役割の言葉で話す。このファイルはそれらの役割を、このリポジトリの issue tracker で実際に使われているラベル文字列にマップする。

| mattpocock/skills 側のラベル | 当リポジトリでのラベル | 意味                                 |
| ---------------------------- | ---------------------- | ------------------------------------ |
| `needs-triage`               | `needs-triage`         | メンテナがこの issue を評価する必要  |
| `needs-info`                 | `needs-info`           | reporter からの追加情報待ち          |
| `ready-for-agent`            | `ready-for-agent`      | 完全に仕様化済み、AFK エージェント可 |
| `ready-for-human`            | `ready-for-human`      | 人間による実装が必要                 |
| `wontfix`                    | `wontfix`              | 対応しない                           |

スキルが役割を言及したら（例: 「AFK-ready triage ラベルを付与せよ」）、この表から対応するラベル文字列を使う。

右列を、実際に使っている語彙に合わせて編集する。
